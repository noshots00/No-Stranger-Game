# Season One QA Checklist (Minimum)

## Character Creation and Identity
- [ ] New player enters adaptive creation flow instead of plain name-only flow.
- [ ] Race reveals adaptively and locks for Season One.
- [ ] Hidden attribute/race progress data persists after reload.

## Idle Progression
- [ ] Daily sim updates health/gold/logs deterministically.
- [ ] XP increases over simulated days and can trigger level-up logs.
- [ ] Autonomous sim never causes character death.

## Main Quest and Side Quest Clarity
- [ ] Main quest remains visible as primary progression path.
- [ ] Rare side quest teasers appear as secondary options.

## Player Quests
- [ ] Post/accept/complete flows still function.
- [ ] Expired quest can be refunded by poster through settlement action.

## Visual and Readability
- [ ] Placeholder location imagery appears in chapter and territory screens.
- [ ] Placeholder portrait appears in quest board cards.
- [ ] Mobile text contrast and readability are improved against dark backgrounds.

## Dev Tools
- [ ] Dev panel exposes Skip +1/+3/+7 day controls.
- [ ] Delta readout appears after skips and reflects deterministic changes.
