# No Stranger Game Thesis and Constitution

## Product Thesis

No Stranger Game is a verifiable single-player seasonal RPG where each crucial choice is:

- signed by the player identity,
- anchored in time,
- and permanently part of that player's path.

The game is designed around consequence rather than completion.

## Core Claims

1. Choice is temporal proof-of-work.
2. Missing branches are legitimate status markers.
3. The world respects player absence and does not punish offline time.
4. Character history should remain portable across clients through Nostr events.

## Constitution (Non-Negotiables)

1. No engagement treadmill:
   - no streak punishments,
   - no grind debt,
   - no fear-of-missing-out pressure loops.
2. Irreversibility must be visible:
   - unchosen options and inaccessible branches should remain legible as absence.
3. Solo-first with social bleed-through:
   - social metadata enriches story but must not block progression.
4. Local-first responsiveness:
   - core interactions complete instantly with local persistence and network publication as follow-up.
5. Consent-first experimental social mechanics:
   - high-risk mechanics are opt-in and reversible.

## Build Gate

Any new feature must answer:

- Does it strengthen signed consequence?
- Does it preserve non-coercive pacing?
- Does it avoid collapsing paths into a solved optimal route?
