# No Stranger Game Nostr-Native Mechanics Tiering

## Objective

Define a build order for the 10 proposed Nostr-native mechanics that maximizes:

- player pull in early launch,
- implementation speed,
- low moderation and abuse risk.

## Tier 1: Immediate Build

These features are approved for first implementation after current MVP stabilization.

1. Echoes (Kind 1 Absorption)
2. Scrying Pool (Followers as Glimmers)
3. NIP-05 True Name / Homeland

## Tier 2: Build After Core Loop Stabilizes

These features are approved after daily quest rhythm and profile/game state are stable.

4. Reactions as Blessings/Curse
5. Relay-as-Difficulty
6. Gray Lady (Relay Geography)

## Tier 3: Policy-Gated Experimental Mechanics

These features require explicit policy, abuse checks, and opt-in controls before implementation.

7. Zap-Powered Quest Influence
8. Mute List as Trauma
9. Kind 1984 as Scars
10. NIP-07 Summoning

## Launch Order Decisions

- Start with low-UI, high-personalization mechanics.
- Preserve the "gentle social layer" principle: no forced interactions in Tier 1.
- Keep economy and social pressure mechanics delayed until abuse guardrails exist.

## Milestone Mapping

- Milestone A (Now): Tier 1 specs and implementation.
- Milestone B: Tier 2 systems behind feature flags.
- Milestone C: Tier 3 policy approval and gated beta tests.
